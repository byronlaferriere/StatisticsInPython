Preview Documentation
Web Servers

Languages like PHP will require you to have a web server set up. If you have not yet done this, please refer to the Box Parts documentation, where you will see how to install Apache or Nginx onto your Box. You can do all this from the Terminal.

Server-Side Applications

Languages like Ruby and Node can run your applciation without the need for a web server, so you should treat this as Dynamic content when reading the Preview Documentation.
